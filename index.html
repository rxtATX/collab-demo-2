What is CSS?

Cascading Style Sheets is a style sheet language that describes the presentation of a webpage. If HTML is the skeleton ofa webpage, then CSS is the meat on the bones. Whereas HTML strictly pertains to webpage markup, CSS deals with colors, aesthetics, and visual layout. It hooks onto specific elements of an HTML page and formats them using any number of style options.
